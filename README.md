# 📊 Análise de Performance — Instagram e TikTok

Análise de dados de redes sociais feita no Google Sheets, cobrindo performance de posts, crescimento de seguidores e melhores dias para publicação no Instagram e TikTok.

## 📁 Estrutura da Planilha

| Aba | Descrição |
|-----|-----------|
| Instagram | Dados brutos do Instagram |
| TikTok | Dados brutos do TikTok |
| Top Tipos Posts (Instagram) | Comparativo de performance por tipo de post |
| Taxa de Engajamento (Instagram) | Taxa de engajamento por tipo de post |
| Crescimento de Seguidores (Instagram) | Evolução de seguidores + ganho diário |
| Top Dias da Semana (Instagram) | Média de alcance e engajamento por dia |
| Visão Geral (TikTok) | Média de visualizações e engajamento por dia |
| Crescimento de Seguidores (TikTok) | Evolução de seguidores + ganho diário |
| Dashboard | Resumo visual com gráficos e insights |

## 🔍 Principais Insights

### Instagram
- **Melhor formato:** Reels (maior alcance médio e engajamento de 1,45%)
- **Melhor dia para alcance:** Sábado (média de 3.457)
- **Melhor dia para engajamento:** Terça-feira

### TikTok
- **Melhor dia para visualizações:** Quarta-feira (média de 34.192)
- **Melhor dia para engajamento:** Domingo
